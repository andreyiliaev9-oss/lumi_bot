# 🤖 ЛЮМИ БОТ
Персональный Telegram-бот помощник для отслеживания привычек, планирования событий, ведения дневника и трекера женского цикла.
## ✨ Возможности
- **👤 Профиль** — система уровней и XP, серии дней, статистика
- **✅ Привычки** — создание привычек с напоминаниями
- **📅 Планировщик** — события с множественными уведомлениями
- **🔒 Личное** — приватный раздел с PIN-кодом защитой
- **🌙 Цикл** — трекер женского цикла с фазами и прогнозами
- **🔔 Центр уведомлений** — единая система управления уведомлениями
- **🌙 Тихий режим** — отключение уведомлений в заданное время
## 🛠 Технологии
- Python 3.11+
- aiogram 3.x
- APScheduler
- SQLAlchemy 2.0 (aiosqlite)
- python-dotenv
## 🚀 Установка
1. **Клонируй репозиторий:**
```bash
git clone <repo-url>
cd lumi-bot
```
2. **Создай виртуальное окружение:**
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```
3. **Установи зависимости:**
```bash
pip install -r requirements.txt
```
4. **Настрой переменные окружения:**
```bash
cp .env.example .env
# Отредактируй .env файл
```
5. **Запусти бота:**
```bash
python bot.py
```
## ⚙️ Конфигурация (.env)
```env
BOT_TOKEN=your_bot_token_from_botfather
ADMIN_ID=your_telegram_id
BOT_NAME=ЛЮМИ
DEBUG=False
DATABASE_URL=sqlite:///lumi_bot.db
```
## 📱 Команды
- `/start` — Запуск бота, главное меню
- `/help` — Справка
- `/admin` — Панель администратора
- `/cancel` — Отмена текущего действия
## 👤 Профиль
Система уровней:
- Начисление XP за активность
- Автоматическое повышение уровня
- Отслеживание серий дней (streak)
## ✅ Привычки
- Создание привычек с описанием
- Напоминания: ежедневно, по дням недели, с повтором
- Отметка выполнения/пропуска
- Статистика и серии
## 📅 Планировщик
- Создание событий с датой и временем
- Уведомления: 24ч, 12ч, 7ч, 5ч, 3ч, 1ч
- Статусы: выполнено, пропущено, перенесено
## 🔒 Личное (PIN)
- 4-значный PIN-код
- 3 попытки ввода, блокировка на 5 минут
- Автовыход через 5 минут бездействия
- Полная приватность записей
## 🌙 Цикл
- Трекер фаз цикла
- Прогноз менструации и овуляции
- Ежедневные записи самочувствия
- Персонализированные советы
## 🔧 Админ-панель
- Рассылка сообщений всем пользователям
- Управление утренними/ночными сообщениями
- Управление советами для цикла
- Просмотр статистики пользователей
